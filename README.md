🐍 Proyecto de Resolución de Katas en Python
📌 Descripción del proyecto
Este proyecto consiste en la resolución progresiva de una serie de 41 ejercicios (katas) en Python, con el objetivo de reforzar los conceptos fundamentales de programación y desarrollar habilidades prácticas en lógica computacional.

🎯 Objetivos
Practicar la sintaxis básica de Python
Comprender estructuras de control (if, elif, else)
Trabajar con funciones y modularidad
Aprender el uso de funciones avanzadas (map, filter, reduce)
Introducir programación orientada a objetos (POO)
Manejar errores y excepciones
Desarrollar pensamiento lógico

🛠️ Tecnologías utilizadas
Python 3
Visual Studio Code
Jupyter Notebook (para pruebas iniciales)

📚 Metodología de trabajo
El proyecto se desarrolló siguiendo un enfoque progresivo:
Lectura del enunciado
Comprensión del problema planteado
Identificación de inputs y outputs
Planteamiento de solución
Descomposición del problema
Selección de estructuras adecuadas
Implementación del código
Escritura de funciones
Uso de buenas prácticas
Pruebas y validación
Ejecución con distintos casos
Corrección de errores
Optimización
Simplificación del código
Uso de funciones avanzadas cuando era posible

🧩 Contenidos trabajados
🔹 Nivel básico
Variables y tipos de datos
Condicionales
Bucles
Funciones
🔹 Nivel intermedio
Listas, tuplas y diccionarios
Comprensiones de listas
Funciones map, filter
🔹 Nivel avanzado
reduce()
Manejo de excepciones (try/except, raise)
Programación orientada a objetos (clases y métodos)

🌳 Ejercicios destacados
✔ Cálculo de frecuencias en texto
✔ Manipulación de listas con funciones funcionales
✔ Cálculo recursivo (factorial)
✔ Creación de clases (Arbol, UsuarioBanco)
✔ Procesamiento de texto dinámico
✔ Validación de datos del usuario

⚠️ Problemas encontrados
Uso incorrecto de input() en ciertos entornos
Dificultad para entender alguno de los enunciados planteados
Errores con self en clases
Dificultades iniciales con map, filter y reduce
Manejo de excepciones
Cálculo recursivo (factorial)
Programación orientada a objetos (clases y métodos) (En este apartados es donde me he encontrado con las mayores dificultades, teniendo que recurrir a recursos extra en internet, como, por ejemplo, otros ejemplos a parte de los proporcionados por el curso)

📈 Aprendizajes clave
Importancia de dividir problemas grandes en partes pequeñas
Relevancia de validar inputs del usuario
Uso correcto de estructuras de datos

🚀 Conclusión
Este proyecto ha permitido consolidar una base sólida en Python, mejorando tanto la lógica de programación como la capacidad de resolver problemas de forma estructurada. Igualmente, considero que todavía debo seguir reforzando muchos de los conceptos clave del lenguaje de programación.
